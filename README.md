This is our graduate project which taking about one nonparametric regression method acting on mixed high dimensional data, including functional, continuous and categorical data: kernel smoothing method. This method is somehow basic unsupervised method which can be used on machine learning.

The experiment is based on HCI HAR dataset where they detecting human behaviour by receiving signal from phones. The theoritical part is from research.

You can download and change the data file directionary on your own computer.
